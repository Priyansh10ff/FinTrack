# 🚀 FinTrack

**Smart Personal Finance & Expense Analytics**

FinTrack is a modern, high-performance personal finance application built with React 19 and Vite. Designed for clarity and efficiency, it provides real-time insights into your financial health through a beautiful, responsive dashboard and interactive visualizations.

---

## ✨ Key Features

- **📊 Dynamic Dashboard**: Get an immediate pulse on your finances with real-time balance, income, and expense tracking.
- **💸 Transaction Management**: Effortlessly record and categorize your spending with full CRUD capabilities.
- **🔍 Smart Filtering**: Find exactly what you're looking for with advanced search and filtering by category, date, or type.
- **📅 Budgeting System**: Set monthly limits and monitor progress with intuitive visual indicators.
- **📈 Interactive Analytics**: Visualize your spending trends and category distribution through dynamic charts and graphs.
- **📱 Responsive Design**: A premium, mobile-friendly interface featuring a streamlined sidebar layout.
- **🔔 Live Notifications**: Real-time feedback via toast notifications for every action.
- **✅ Robust Validation**: Secure data entry powered by React Hook Form and Yup.

---

## 🛠️ Tech Stack

- **Core**: [React 19](https://react.dev/), [Vite](https://vitejs.dev/)
- **Routing**: [React Router Dom](https://reactrouter.com/)
- **State Management**: [React Context API](https://react.dev/reference/react/useContext)
- **Data Visuals**: [Recharts](https://recharts.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: Vanilla CSS (Modern Design System)
- **Forms**: [React Hook Form](https://react-hook-form.com/), [Yup](https://github.com/jquense/yup)
- **Utilities**: [Axios](https://axios-http.com/), [Date-fns](https://date-fns.org/), [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Priyansh10ff/FinTrack.git
   cd FinTrack
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```

4. **Build for Production**
   ```bash
   npm run build
   ```
